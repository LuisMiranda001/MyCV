Luna Cockpit – Access Request & Approval Automation Platform

Enterprise service request solution built with Power Apps and Power Automate to manage access approvals, validations, and Active Directory group memberships.


Business Context

Project Summary


📌 Project context

Luna Cockpit was a project fully led and developed by me as an internal solution for the Brazil team, focused on centralizing and automating service requests, with special emphasis on access requests.

Before this solution, access approvals were handled manually through emails and informal follow‑ups, resulting in delays, limited visibility, and poor traceability for both users and approvers.


🛠️ Implemented solution

Luna Cockpit was built using Power Apps, structured with multiple menus. The core module was the Access Request feature.
Users submit the required information to request access, which is then stored in a SharePoint List. This action triggers a Power Automate flow that manages the entire ticket lifecycle automatically.
The solution includes validations, approvals, escalations, notifications, and status tracking with minimal manual intervention.

In addition to access requests, Luna Cockpit includes dedicated modules for personalized communication, user feedback, and a Learning Hub.
- The Contact module allows users to send structured messages to the corresponding support teams, including attachments and predefined routing.
- The Feedback module collects user satisfaction and comments to continuously improve the platform.
- The Learning Hub centralizes documentation, training materials, and user guidance related to Luna and other performance dashboards.


🔧 Key Learnings

This project helped to:

  • - Automate access governance workflows
  • - Improve traceability and user experience
  • - Reduce approval times and manual intervention
  • - Lead and deliver an enterprise solution independently
  • - Continuous improvement through structured user feedback
  • - Integrate Power Apps, Power Automate, SharePoint, and Active Directory into a unified system
